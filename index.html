Financial web

Financial web es un pequeño modulo financiero hecho con javascript puro, en el cual se pueden calcular intereses para un prestamo a determinado tiempo. Además muestra una tabla de amortización con información relevante del prestamo.

Login

Login hecho en angular, acompañado del proveedor de servicios firebase en el cual tenemos un sistema de autenticación de usuarios. Además nos permite confirmar el usuario mediante el email, solicitar un cambio de contraseña entre otras funciones.

APP rick & morty

Hecho con react js, la App rick & morty recopila datos de la API publica Rick And Morty y mostrando en la vista la información requerida de los personajes de esta serie.

Movies

Un buscador de peliculas, está realizado con angular v14, recolectando información de una API de peliculas y mostrando la información solicitada de cada pelicula.
